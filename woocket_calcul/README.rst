Woocket Calculator
==================

Woocket Calculator adds a simple calculator directly in the Odoo systray
(top navigation bar), allowing users to perform quick calculations
without leaving the current screen.

Features
--------

- Calculator available from the systray
- Quick access without leaving the current screen
- Clean and lightweight user interface
- Compatible with Odoo 19.0

Usage
-----

After installation, a calculator icon appears in the Odoo top bar.
Click the icon to open the calculator and start calculating.

Technical Information
---------------------

- Backend-only module
- No database models
- No configuration required

Compatibility
-------------

- Odoo 19.0
